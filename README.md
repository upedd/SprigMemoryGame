# Sprig Memory Game
Memory game written for [Sprig Console](https://sprig.hackclub.com/)

**Open it in sprig editor**: [click](https://sprig.hackclub.com/gallery/memory_game)

## Instructions
Pair matching cards to earn score.

Controls:
`wasd` to move, 
`i` to interact
